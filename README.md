# GenLayer Protocol: Security Audit & Strategic Enhancements
**Contributor:** Affcryptohunt
**Task:** Research & Analysis (100-2000 pts)

## Overview
This repository contains a deep-dive security audit of the GenLayer Intelligent Smart Contract (ISC) framework, focusing on non-deterministic consensus risks and LLM-specific attack vectors.

## Key Findings
1. **The "Sanity Floor" Vulnerability:** Analysis of how malicious web data can bypass simple scraper logic.
2. **Consensus Bias:** Risks associated with validator model homogeneity.
3. **Prompt Injection:** Demonstrated hijacking of contract logic via external web content.
