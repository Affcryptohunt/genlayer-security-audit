Proposal: Multi-Model Consensus (MMC)
Problem: If all validators use GPT-4, they share the same "hallucination surface."
Solution: Mandate that a "Jury" must consist of at least three distinct model architectures (e.g., 1x OpenAI, 1x Anthropic, 1x Meta-LLaMA).
Metric: Implement a "Diversity Score" on-chain. Validators with unique model signatures receive a 1.2x reward multiplier for contributing to high-entropy consensus.
